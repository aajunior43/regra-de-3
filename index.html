Calculadora de Regra de Três

Para usar esta calculadora, preencha os três primeiros campos com os valores conhecidos da sua proporção. Por exemplo, se você sabe que 2 está para 4, assim como 3 está para X, você preencheria os campos com 2, 4 e 3. Depois, pressione 'Calcular' para obter o resultado.
